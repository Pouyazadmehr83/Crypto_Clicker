## ✨ ویژگی‌ها

-   **مکانیک ساده:** با کلیک کردن سکه جمع آوری کنید
-   **سیستم ارتقاء:** آیتم‌های مختلف برای افزایش درآمد
-   **درآمد خودکار:** دارایی‌هایی که حتی در حالت غیرفعال درآمد تولید می‌کنند
-   **رابط کاربری ساده و روان**
